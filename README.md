# Fiverr Filla 🚀

A Chrome extension that uses **Groq AI** to generate and autofill your entire Fiverr presence — gig pages, seller profile, work experience, bio, and skills — all from one click.

## ✨ Features

### Gig Editor
- **AI Title** — Compelling "I will..." titles under 80 characters
- **Auto Tags** — 5 relevant gig search tags
- **3-Tier Packages** — Basic / Standard / Premium with unique names and realistic prices
- **Formatted Description** — Question hook, intro, bullets, and CTA
- **5 FAQs** — Buyer-voiced questions with personal answers
- **Buyer Requirements** — Auto-fills and marks required
- **Per-Gig Niche Input** — Niche bar injected directly on each gig page

### Seller Profile
- **Bio / About** — Professional seller bio with personalized details
- **Work Experience** — Full entry auto-filled and submitted
- **Skills** — Picks 6 relevant skills from Fiverr's database

### General
- **Stop Button** — Abort mid-task with one click
- **Human-like Typing** — Character-by-character input with natural delays
- **Multi-key Rotation** — Up to 3 Groq API keys, auto-rotated on rate limit
